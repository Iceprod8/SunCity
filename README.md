# SunCity – Guide rapide

## Structure du projet

- `backend/` : API mockée avec **json-server**.
- `frontend/` : application **Angular**.

## Lancer le backend (API json-server)

```bash
cd backend
npm i
json-server db.json
```

## Lancer le frontend Angular

```bash
cd frontend
npm i
npm start
```

L’application est disponible sur `http://localhost:4200`.
